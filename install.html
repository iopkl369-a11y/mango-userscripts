더망고 유저스크립트 설치

팀 공용 발주 자동화 스크립트. 아래 순서대로 한 번만 설치하면 됩니다. 이후 업데이트는 자동.

1. Tampermonkey 설치 (브라우저별 1회)

쓰는 브라우저에 Tampermonkey 확장을 설치하세요.
Chrome · Edge · Brave Firefox Safari

2. 스크립트 설치 (링크 클릭 → "설치")

각 링크를 누르면 Tampermonkey 설치 화면이 뜹니다. 설치를 누르세요. 브라우저 분리 규칙에 맞춰 필요한 것만 설치.

스크립트설치 브라우저설치
⚙ 팀 설정 (필수·먼저)쓰는 브라우저 전부설치
간단입력 + 한방캡처 (Opt+W/Q)전부설치
무신사 주소입력 (Alt+A)Edge설치
SSG 주소입력 (Alt+A)Brave설치
네이버 주소입력 (Alt+A)Chrome / Brave설치
무신사 주문완료 광고 스킵무신사 쓰는 브라우저설치

3. 팀 설정값 입력 (브라우저별 1회)

망고 사이트(더망고·무신사·SSG·네이버) 아무 페이지나 열면 좌하단에 ⚙ 망고설정 버튼이 보입니다.
클릭 → 팀 채팅으로 받은 설정 JSON을 붙여넣고 저장. (코드표·회사번호 — 이 값은 공개되지 않고 각자 브라우저에만 저장됩니다)
⚠ 설정값(실명·번호)은 공개 저장소에 없습니다. 반드시 팀 채팅으로 받은 JSON을 붙여넣어야 코드표·회사번호가 동작합니다.

업데이트

스크립트가 개선되면 Tampermonkey가 자동으로 가져옵니다(수동 재설치 불필요). 즉시 받으려면 Tampermonkey 대시보드 → 유틸리티 → "스크립트 업데이트 확인".

안 될 때

  • 설치 링크가 그냥 텍스트로 열리면: Tampermonkey가 아직 설치 안 됨(1단계) 또는 비활성.
  • 좌하단 ⚙ 버튼이 ⚠ 표시면: 설정 JSON 미입력(3단계).
  • 단축키 Mac=Option, Windows=Alt.
